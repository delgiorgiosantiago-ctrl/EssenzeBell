# 🌿 EssenzeBell

## 🧼 Descripción
**EssenzeBell** es una aplicación web desarrollada para la visualización de un **local de productos de limpieza**.

Funciona como un **catálogo online**, donde los usuarios pueden explorar productos y comunicarse directamente con el negocio mediante WhatsApp para concretar la compra.

> ⚠️ Actualmente no cuenta con compras online integradas.

---

## 🏪 Funcionalidades principales

### 📦 Catálogo de productos
- Listado de productos disponibles
- Organización por categorías
- Detalle de producto (nombre, precio, descripción)

### 💬 Consulta vía WhatsApp
- Botón de contacto directo
- Redirección automática a WhatsApp con mensaje predefinido

### 📊 Gestión interna
- Administración de productos
- Actualización de stock y precios

---

## 🛒 ¿Cómo comprar?
1. El usuario navega el catálogo  
2. Selecciona un producto  
3. Hace clic en el botón de contacto  
4. Es redirigido a WhatsApp para realizar la consulta o compra  

---

## 🧰 Tecnologías utilizadas
- **Backend:** Laravel  
- **Frontend:** Blade + Bootstrap  
- **Bundler:** Vite  
- **Base de datos:** MySQL / SQLite
